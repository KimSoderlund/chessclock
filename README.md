# chessclock
This is a chessclock ment as a last resort for tabletop wargaming when no ordinary chessclock is available.
It's supposed to be used on a cellphone with sleep mode off to avoid the screen shutting down.

## Features
Big glowing half screen buttons, making it easy to use and see who is the active player.
You can set the play time and when "low time alert" kicks in for both players.
The application stores the current player times in local storage, so that the time doesn't reset if the page isn't active or shut down.

### Upcoming Features
Set individual times for each player (for fringe scenarios when only one player needs to change how much time they have)
